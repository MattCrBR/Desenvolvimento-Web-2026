- Introdução a HTML

## Versionamento e Deploy

O versionamento, também conhecido como controle de versões, é essencial no desenvolvimento web.

### Benefícios do Versionamento

- Recuperar versões específicas  
- Facilitar colaboração em equipe  
- Permitir ramificações (branches)  

### Outros tópicos

- Conceito de deploy  
- Ambientes de desenvolvimento  
- Plataformas: GitHub, GitLab e Bitbucket  
- Integração Contínua (CI/CD)  
- Automação e boas práticas

## Introdução ao JavaScript (Vanilla JS)

### O que é

Linguagem de script interpretada,essencial para o desenvolvimento front-end.

### Vanilla JS

Refere-se ao JavaScript puro, sem bibliotecas externas, base para todas as abstrações.

### Importância

Permite:

- Manipulação de páginas  
- Comunicação com servidores  
- Implementação de lógica de interface

## Website Builders

### Vantagens

- Facilidade de uso com interfaces intuitivas 
- Rapidez no desenvolvimento  
- Bom custo-benefício com planos acessíveis 
- Templates prontos com design profissional e responsivo  

### Limitações

- Flexibilidade limitada com dificuldade em implementar funcionalidades muito específicas
- Possível perda de performance por conta de alguns ter código mais pesado 
- Migração pode ser complexa caso seja necessário migrar para outra plataforma

## 📊 Critérios de Avaliação

- **AC1 - 15%:** Prova escrita de múltipla escolha, com questões elaboradas para verificar a compreensão teórica e a aplicação dos conceitos discutidos em aula.
- **AC2 - 30%:** A avaliação será baseada na entrega e apresentação do projeto, com foco no processo de gestão adotado, incluindo a organização do backlog, as páginas desenvolvidas, e a análise de benchmarking realizada  
- **Avaliação Geral - 10%:** Prova online com 40 questões realizada no dia 04/05  
- **Avaliação Final - 45%:** A avaliação final será composta pela documentação completa do projeto, que deve ser elaborada como um “mini TCC”.
